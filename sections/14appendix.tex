\chapter*{Appendix A: Complete requirements.txt}
\addcontentsline{toc}{chapter}{Appendix A: Complete requirements.txt}

The following Python packages are required to run An Intelligent Hybrid Machine Learning Framework for UPI Fraud Detection. Install them using \texttt{pip install -r requirements.txt} inside a Python 3.10 virtual environment.

\begin{lstlisting}[language=bash, caption={requirements.txt -- Python dependencies}]
pandas
numpy
scikit-learn
xgboost
networkx
imbalanced-learn
fastapi
uvicorn
streamlit
requests
shap
plotly
\end{lstlisting}

  "amount": 48000,

  "receiver_state": "Delhi",
  "sender_bank": "HDFC",

\chapter*{Appendix C: Project Repository Structure}

The complete project repository is publicly available at:

\url{https://github.com/smrutiranjanbhuyan/upi-guard}

\begin{verbatim}
upi-guard/
|
|-- data/
|   +-- upi_100k_ultra_realistic.csv     (Dataset: 100K records)
|
|-- model/                                (Generated after training)
|   |-- fraud_model.pkl
|   |-- feature_columns.pkl
|   +-- threshold.pkl
|
|-- src/
|   |-- preprocess.py                    (Feature engineering)
|   |-- train.py                         (Model training)
|   |-- optimize_threshold.py            (Threshold optimization)
|   |-- explain.py                       (SHAP explainability stub)
|   +-- upi_fraud_detection_pipeline.ipynb (EDA notebook)
|
|-- api/
|   +-- main.py                          (FastAPI REST endpoint)
|
|-- ui/
|   +-- dashboard.py                     (Streamlit dashboard)
|
|-- requirements.txt
|-- .gitignore
+-- README.md
\end{verbatim}
